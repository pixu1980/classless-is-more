Attribute utilities instead of classes

Do you mean using attributes instead of classes? Yep!

Do you know CSS attr() function is about to get a game-changer update?

What is attr()?

      
        

      
    
      
        div::after {
          content: attr(data-title);
        }
Traditionally limited to the content property and returned strings, the attr() function in CSS allows you to retrieve the value of an HTML attribute and use it in your styles.

Our HTML is already
full of attributes 

      
        <button disabled aria-label="Close modal">x</button>
        <input type="email" required placeholder="Your email" />
        <div data-theme="dark" data-size="lg"></div>

Why not use them
for CSS selectors? 

      
        [aria-label] { ... }
        [type="email"] { ... }
        [data-size*="lg"] { ... }

CSS attr() evolution

Unlocking infinite possibilities with typed attributes

  • Content-first web design
  • Accessible and functional by default
  • Declarative, readable, scalable
  • No class spaghetti
  • Easy to debug and maintain
  • Reduced dependency on JS frameworks

What's New?

Now, attr() supports typed values, enabling its use beyond content property

      
        div {
          font-size: attr(data-font-size length);
          color: attr(data-color color, black);
        }

Supported types include: angle, color, custom-ident, image, integer, length, length-percentage, number, percentage, resolution, string, time, transform-function, ...

Example: Grid Control

Use attr() to control grid layout via HTML attributes.

      
        <div data-column-start="1" data-column-end="3">
          this is the div content
        </div>
      
    
      
        div {
          grid-column-start: attr(column-start integer);
          grid-column-end: attr(column-end integer);
        }

Example: Dynamic Colors

Set colors dynamically using attributes.

      
        <div data-color="rebeccapurple">Red Text</div>
      
    
      
        div {
          color: attr(data-color color, black);
          background-color: color-mix(in srgb, attr(data-color color) 50%, white 50%);
        }

Example: Font Sizes

Control font size through attributes.

      
        <div data-font-size="1.5rem">Large Text</div>
      
    
      
        div {
          font-size: attr(data-font-size length);
        }

Custom Properties Integration

Pass attribute values to custom properties for broader usage.

[data-button-color] {
--button-color: attr(data-button-color color);
}

button {
background-color: var(--button-color);
}

HTML:

<div data-button-color="#00ff00">
<button>Click Me</button>
</div>

Why Use data-* Attributes?

  • Prevents conflicts with standard attributes
  • Provides a clear namespace for custom data
  • Accessible via JavaScript through dataset

Browser Support

Typed attr() is currently supported in Chrome and Edge. Support in other browsers is forthcoming.

Always check current compatibility before using in production.

Conclusion

The enhanced attr() function expands CSS capabilities, allowing for more dynamic and maintainable styles directly tied to HTML attributes.

Embrace this feature to create more flexible and semantic designs.
