Does it makes sense resetting CSS in 2025?

In modern web dev, user agents are frontline accessibility framework.

If we write semantic HTML, gently extending it with CSS, we inherit decades of research about assistive technologies support by default.

This isn't a fallback.
It's the starting line.

What are the main concerns with CSS Resets?

Concern #0: CSS Resets can be harmful to users who rely on keyboards for navigation

Concern #1: Every element that is “zeroed” must then be redefined

Concern #2: CSS Resets are technical and design debt at the same time
